Por que o GPS erra?

O GPS não recebe sinais perfeitos. Diversos fatores podem introduzir erros:

  • Atmosfera terrestre
  • Prédios altos
  • Montanhas
  • Condições climáticas
  • Ruído eletrônico
  • Reflexão de sinais

Filtro de Kalman

O Filtro de Kalman é um algoritmo matemático utilizado para estimar valores reais a partir de medições com ruído.


Ele é amplamente utilizado em:

  • GPS
  • Drones
  • Aeronaves
  • Robôs
  • Satélites

Seu objetivo é combinar medições sucessivas para produzir uma estimativa mais precisa da posição real.
